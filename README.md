# simple php event loop ( select system call / non blocking io)

```
*** SOCKET EXTENSION SHOULD BE ENABLED IN PHP.INI FILE ***
```
```
*** WINDOWS IS NOT SUPPORT ***
```
**first start the server**
```php
php server.php
```
**then run client script as many as you want**
```php
php client.php
```

    setInterval
    setTimeout
